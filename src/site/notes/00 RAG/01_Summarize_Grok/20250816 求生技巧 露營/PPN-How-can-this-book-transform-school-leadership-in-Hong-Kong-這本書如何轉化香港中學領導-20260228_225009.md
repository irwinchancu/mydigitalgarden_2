# PPN- How can survival skills build crisis-ready school leadership_ 生存技能如何鍛造危機就緒校領？


## 1. 核心洞見
- **預防勝於治療，危機領導從預知症狀開始**：失溫症分級教導我們，教育領導須及早辨識團隊「輕症」疲勞，如師生士氣低落，避免惡化成「重症」崩潰。學校如野外，助理校長需建立症狀監測系統，轉化為日常健康檢查機制，確保組織體溫維持穩定。

- **資源即興利用，轉化日常廢物為生存利器**：從寶特瓶鞋到紙箱馬桶，書中展現創意重用，領導視野下，這是學校預算有限時的啟發—廢紙、膠帶變急救工具。哲思在於「無中生有」，培養師生視野，從資源匱乏中發掘潛力，建構自給自足校園文化。

- **技能鍛鍊體力與心力，領導韌性源於實戰**：鑽木取火需1分鐘內成功，強調體力訓練與堅持，助理校長應用於領導—模擬演練地震333守則，3秒保命、3分鐘冷靜，轉化為危機決策框架，提升全校抗壓能力。

- **知識儲備決定存活期，1個月備糧哲學**：水食儲備從3天延至1月，教育領導借鏡，建立「知識糧倉」—防災手冊、技能工作坊，讓學校從短期應急升級長期自立，避免依賴外部救援。

- **團隊分工放大效能，2人取火勝獨行**：弓鑽法需合作，領導洞見是分權賦能—助理校長非萬能英雄，而是設計角色分工，如老師帶生火、學生採野草，模擬校園團隊在災時互補，強化集體領導力。

- **方向感來自天象，領導需永恆北極星**：用星星月亮定位北方，哲思延伸至學校願景—助理校長定「北極星」核心價值，無論風雨，導引師生方向，避免迷失於日常瑣事。

- **信號求援不放棄，反射光達公里外**：CD信號鏡持續傳送，教導領導在危機中主動發聲—學校用故事分享、數據反射，吸引家長社群支持，擴大影響力直至救援到達。

## 2. 行動建議
### 給自己（個人思維層面）
從「生存思維」升級領導框架：視學校為「野外環境」，每日自問「若地震3秒後，我保命動作ready嗎？」轉變心態，預視風險如失溫症狀，建個人危機日誌，記錄團隊「體溫」指標。擁抱「資源黑客」視野，廢物變工具，決策時優先即興方案，培養「1分鐘取火」速度感，拒絕拖延。

### 給日常實踐（立即可執行的行動）
1-3天內，買百圓店★物品（鋁毯、哨子、膠帶），放個人緊急包，模擬失溫處置演練—裹毯躺墊子，教1位老師。1週內，帶團隊河邊找打火石、採蒲公英，煮野草粥分享，記錄心得。每日晨會分享1項333守則（如3分鐘冷靜），用CD練信號鏡，定位學校「北方」目標。

### 給長期系統（建立習慣或流程）
設計「校園生存月曆」：每月1日演練生火/釣魚，追蹤參與率>80%；建紙箱馬桶站，融入防災日。設立「資源庫」，教師捐廢物DIY工具，季度審核使用率。推「北極星導航」APP，師生上傳危機故事，反射光效追蹤影響（如家長參與+20%），複利成全校韌性文化，每年模擬1個月自立挑戰，助理校長領航，溫暖前行。
