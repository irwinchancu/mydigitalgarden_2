# PPN- How can data expose and fix Hong Kong's stalled education reform? 數據如何揭弊並重啟香港教改？


## 1. 核心洞見
- **教改理想淪為應試陷阱**：2000年起推行全人發展、OLE及其他學習經歷，本意減壓多元，但現實中TSA、SBA催生補課操練，學生壓力爆煲，自殺率升。全人教育變全方位競爭，助理校長須反思：數據顯示選科萎縮、應用學習報讀僅7-8%，精英主義阻礙多元出路，領導需轉向數據導向檢視政策盲點。

- **教育資源嚴重落後國際**：OECD數據揭香港教育經費僅GDP 3.3%，排全球128，遠低丹麥7.9%、芬蘭6.5%；師生比高於經合平均，小班教學落空，縮班殺校加劇教師流失。哲思上，這非「不能」而「不為」，領導視野須倡議增加開支，否則全人發展永成空談。

- **新高中學制超載師生**：4+3選修、通識SBA理想融會知識，卻因大學學位不足、試卷深淺不均，致退修急增、科學教育失落、中文成「死亡之科」。數據證明69%考生僅達基本門檻，2萬弱勢生無出路；學校領導需洞察：操練文化非天生，乃政策失衡，須建數據追蹤系統減負。

- **人口炸彈與校網失衡**：少子化下縮班殺校逼加班加派，基層區教師人心惶惶，私立副學位淘汰賽開打。長期視野：一條龍辦學、小班彈性雖可行，但無資源支持只增社會成本；助理校長應以數據預測校網，推動因材施教，避免精英教育加劇不公。

- **文化衝突阻教學轉型**：傳統「傳道授業」 vs 改革「促進者」角色，互動教學欠前備知識易浪費時間；芬蘭成功靠中產平等、無名校文化、優質師資。香港填鴨應試盛行，批判重於建設，領導哲思：數據顯示男生通識劣勢、創意欠缺，需平衡講授與活動，重建樂學文化。

- **多元出路缺失酿社會怨氣**：忽略職業教育，大學泡沫化致基層勞工短缺、畢業生向下流；直資高費利精英，窒礙流動。數據警訊：應用學習僅5000報讀，領導須視為系統危機，倡議IB公校、實用課程，建構「構建多元出路」框架。

- **管理主義與家長壓力雙重夾擊**：監控制肘學校，非教學工作暴增，家長「望子成龍」催谷名校。哲思深度：教改眼高手低、項目雜亂，助理校長需數據說故事，遊說政策調整，壓制操練風氣，聆聽前線。

## 2. 行動建議
### 給自己（個人思維層面）
從數據視角重塑領導框架：視教改失落為「系統性失衡」而非個人失敗，培養「data-driven mindset」，每日問「這政策有數據支持嗎？」。內化芬蘭哲學——樂學優先於競爭，轉變「精英篩選」為「因材施教」認知，自我對話：如何用資源不足轉化為彈性創新？這將升級你的決策韌性，讓你成為學校轉型的哲學家領袖。

### 給日常實踐（立即可執行的行動）
- **今天起建數據儀表板**：收集TSA/SBA/ret修率，1小時內用Excel追蹤班級師生比與壓力指標，與教師分享，減低操練1/3。

- **明日召前線會議**：用OECD表二對比經費，10分鐘遊說教師團隊，優先識別3個高壓科目（如通識），試行「講授+互動」混合課，測量學生滿意度。

- **3天內訪家長**：抽10位家長訪談「望子成龍」壓力，用數據解釋多元出路，推應用學習試堂，記錄反饋追蹤報讀意欲。

- **7天內減負試驗**：削SBA任務20%，替以小組探究，量化退修變化，鼓勵教師記錄「樂學時刻」。

### 給長期系統（建立習慣或流程）
- **數據治理委員會**：每月檢視校內大數據（經費/師生/成績），設KPI如「應用學習報讀升20%」，連結獎勵，複利建多元文化。

- **校網彈性機制**：與鄰校聯盟推一條龍試點，追蹤3年人口數據，自動調整班級/師資，防縮班危機；年檢芬蘭模式，培訓「促進者」教師。

- **政策倡議循環**：季度向局方提交數據報告（e.g. GDP經費比較），建校友/家長聯盟遊說增資；習慣「前線意見日」，融合日本/IB元素，5年內轉型為「樂於學習」學校，追蹤社會流動指標如畢業生就業率。堅持下去，你將見證教育復興的複利力量！
