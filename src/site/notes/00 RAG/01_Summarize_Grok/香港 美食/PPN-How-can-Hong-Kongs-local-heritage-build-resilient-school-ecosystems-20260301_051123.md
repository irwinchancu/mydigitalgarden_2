# PPN- How can Hong Kong's local heritage build resilient school ecosystems? 香港本土遺產如何建構韌性校園生態？


## 1. 核心洞見
- **多功能社區空間的領導啟示**：香港仔市政大廈從街市到圖書館、體育館，一棟建築統合民生、教育、康樂，展現資源高效整合。對學校領導，這提醒我們將校園轉化為社區樞紐，融合學習、運動、文化，避免單一功能浪費。助理校長可借鏡，設計彈性空間，提升學生全人發展與家長參與。

- **本土簡樸養成韌性心態**：蒲台島「香港南極」的海天一色與坤記士多的人情味，彰顯簡樸生活中的堅韌與人際連結。教育領導視野下，這哲思強調擺脫都市浮躁，回歸根源培養學生抗逆力。學校可透過本土故事，建構「少即是多」的校園文化，強化內在動機。

- **傳統工藝傳承文化認同**：煮綠豆沙的浸洗、落臭草、撈殼步驟，細膩展現客家智慧與在地水源運用，象徵世代相傳的社區智慧。領導者應視此為文化教育藍圖，將傳統融入STEM或德育，助學生連結香港身份。長期，這能對抗全球化同質化，培育本土自豪感。

- **生態人文融合的永續視野**：谷埔客家村的海山交織、白鷺彈塗魚生態，與圍屋傳統共存，揭示自然與人文的平衡之道。學校領導可借此推動戶外學習，整合環境教育於課程，培養學生的生態素養與社區責任。這種視野轉化為行動，建構可持續校園模式。

- **偏遠人情的領導力量**：坤記士多平日僅兩班船卻守望相助，賣的不是商品而是「人情」，這是基層領導的哲學核心。助理校長可內化為關懷式管理，重視師生小連結，放大社區影響力。最終，這養成信任生態，讓學校成為「士多般」的溫暖補給站。

- **歷史脈絡賦能未來策略**：從明朝鄭和圖到清朝天后廟，蒲台島記錄香港南端故事，提醒領導者根植歷史前瞻未來。教育應用上，鼓勵學校發掘在地歷史，設計跨學科項目，提升學生策略思維與文化自信。

## 2. 行動建議
### 給自己（個人思維層面）
重新框定領導心態：每日反思「蒲台簡樸」，問自己「今日行動是否回歸教育本質？」內化「人情領導」，練習傾聽一師生故事，轉化為決策同理心。升級認知框架，將校園視為「市政大廈式樞紐」，預想多功能整合機會，養成本土視野的策略直覺。

### 給日常實踐（立即可執行的行動）
1-3天內，組織「在地一日」校會，分享蒲台島或谷埔故事，邀學生畫「我的香港南端」（30分鐘）。明日巡堂時，訪談3位老師，探討如何將傳統工藝如綠豆沙融入家課。週內發校園通訊，推介附近社區設施如公共圖書館，鼓勵師生週末探訪並記錄心得。

### 給長期系統（建立習慣或流程）
設計「本土遺產月曆」：每季選一地點（如蒲台、谷埔），建跨組課程模組，整合歷史、生態、工藝，追蹤學生參與率與反饋。設立「社區士多基金」，每月撥款支持師生互助項目，KPI為連結事件數。導入「生態足跡日誌」App，學生記錄校園綠化行動，複利成年度文化節，持續監測校園認同指數。
