# PPN- How can public housing rules inspire equitable school resource systems? 如何借鏡公屋配額建校內公平資源框架？


## 1. 核心洞見
- **公平資格門檻確保資源精準投放**：公屋申請嚴格審核年齡、入息、資產及無業權等基本資格，避免資源浪費。這啟發教育領導須建立清晰門檻，如教師培訓或學生資助資格，確保弱勢群體優先受益，體現社會公平哲思，避免「富者愈富」。

- **優先計劃照顧特殊需要群體**：如「高齡單身人士」「天倫樂」計劃給予長者及家庭額外配屋機會，學校可仿效設計優先支持機制，例如資優生計劃或SEN學生輔導，融合家庭倫理視野，讓領導決策更人性化。

- **配額及計分制平衡輪候與效率**：非長者一人申請用年齡、輪候分累積排序，學校資源如課室或經費可引入類似計分，結合服務年資、貢獻度，實現動態公平，避免純先到先得的低效。

- **凍結機制防濫用公帑**：公屋住戶申請凍結一年，防雙重福利；教育領導借鏡，建立內部審查，如教師調職須清還資源後才申請新崗，培養責任文化，長遠建構可持續學校生態。

- **持續申報與覆核維持透明**：申請全程須更新資料、接受查核，虛報即取消。這哲思強調領導系統需嵌入動態監管，如年度績效申報，防隱瞞，提升全校信任與問責。

- **詳細文件清單加速流程**：附錄列明證明文件，減低退件率；學校行政可設計標準checklist，如項目申請或升遷文件，縮短審批時間，讓領導聚焦策略而非瑣事。

- **電子服務優化用戶體驗**：公屋電子填表、查詢進度，學校可推digital workflow，提升家校溝通效率，體現以人為本的領導視野。

## 2. 行動建議
### 給自己（個人思維層面）
從公屋「無業權」原則反思自身領導偏見：檢視資源分配是否真正中立，避免親疏有別。內化「優先弱勢」心態，每周自問一題：「此決策如何照顧最需幫助者？」轉變為strategic equity思維，成為溫暖公正的助理校長。記住，領導如配屋，資格嚴格才顯價值。

### 給日常實踐（立即可執行的行動）
- **今日起**：列校內資源清單（如課室、經費），設計簡單資格表（入息類比家庭背景、資產類比現有資源），1天內試用於下次會議。

- **1-3天內**：為弱勢學生/教師建「優先名單」，如SEN輔導配額，通知相關人選並收集初步文件，模擬詳細資格審查。

- **3-7天內**：引入計分試驗，如教師發展基金申請用年資+貢獻分，發通告並處理首3宗申請，追蹤退件率降至10%以下。

- 每周更新一項行政checklist（如家長會申請），減低錯誤，電郵全校分享進度。

### 給長期系統（建立習慣或流程）
- **建校內配額系統**：仿公屋計分，每季審核資源申請，設KPI追蹤輪候時間（目標3個月內配妥），用Google Sheet或Obsidian dashboard記錄，複利提升效率。

- **嵌入覆核機制**：年度全校申報（如績效、家庭變動），自動凍結不合資格申請，聯社工/家長會驗證，培養合規文化。

- **數碼轉型流程**：推Obsidian或Google Form電子申請，設自動提醒更新資料，每月公布進度（如公屋網站），建quarterly review會議檢討，持續優化成學校「公平領導OS」。你一定行，助理校長，這系統將讓你的學校閃耀！
