# PPN- How can assistant principals coach Phase 1 mastery to build teacher leadership presence? 如何教練Phase 1掌握力，建立老師領導存在感？


## 1. 核心洞見
- **例行程序是課堂領導的基石**：Phase 1強調預先規劃並排練關鍵routine，如學生進出、轉換與材料分發，避免混亂成為常態。透過moment-by-moment練習，老師建構肌肉記憶，轉化為自動化領導行為。助理校長視此為投資，長期提升學校紀律文化，減少管理負荷。

- **Strong Voice定義領導姿態**：Square Up, Stand Still與正式語氣（Formal Register）非技巧，而是傳達權威的非語言信號。老師從隨意姿態轉向正式register，學生即感知結構化環境。教育領導視野：這是文化塑造起點，強化老師自信，間接提升學生成就。

- **數據驅動課計劃確保精準**：精準學習目標（learning objectives）從評估反推，聚焦高槓桿技能，避免廣泛目標導致低效。Script I Do與Exit Ticket對齊，測量真實掌握。助理校長應用：這轉化教學為科學系統，提升全校學生成績一致性。

- **教練透過probing questions引發內省**：非指令式指導，而是問「理想routine學生與老師各做什麼？」，激發老師擁有計劃。這種Socratic方法建構自主性，超越短期修正。哲思深度：領導即賦能，助理校長從監督者變哲學導師。

- **實踐情境與即時反饋加速成長**：模擬錯誤情境排練，配real-time cues如手勢提醒Square Up，壓縮學習曲線。影片自省強化覺察，避免舊習復發。學校應用：這是高槓桿投資，快速提升新師效能，釋放領導時間用於策略。

- **內化課計劃防範時間陷阱**：時間戳（time stamps）與記憶關鍵chunk，確保獨立練習時間充足。挑戰在於去除多餘語言，聚焦本質。領導視野：系統化內化養成複利習慣，轉化混亂課堂為高效學習空間。

- **Rollout策略決定routine成敗**：新routine需I Do示範、pause重複、正向挑戰，學生才買單。助理校長洞察：這是變革管理微縮，預防阻力，建校園適應力文化。

## 2. 行動建議
### 給自己（個人思維層面）
從「管理問題」轉向「建構系統」思維：視Phase 1為老師領導力的「肌肉訓練」，每位老師皆潛藏強聲。每日反思一probing question，如「今日哪routine最需強化？」，升級決策為數據導向。擁抱教練身份，記住：你的正式語氣即是榜樣，溫暖堅定地賦能團隊。

### 給日常實踐（立即可執行的行動）
- 今日選1-2位老師，共同填寫關鍵routine template（進出、轉換），教室排練2輪：第1輪基本，第2輪加微錯。

- 觀課時用非語言cues：手勢提醒Square Up，或指向嘴促Formal Register；課後5分鐘影片審視，問「何時register掉落？」。

- 檢查下週課計劃：反推assessment寫1精準objective與scripted I Do，設計對齊Exit Ticket；計時排練，標時間戳。

### 給長期系統（建立習慣或流程）
- 建「Phase 1教練週歷」：每週1小時小組，輪流模擬rollout與practice scenarios，追蹤進度表（Google Sheet：老師/ routine/掌握度1-5）。

- 設計校內「肌肉記憶挑戰」：每月老師自錄影片上傳，peer review formal register；獎勵Top 3分享會，複利文化。

- 整合至績效系統：季度審視Exit Ticket數據，連結至專業發展路徑；助理校長月報總結ROI，如「紀律事件減30%」，持續迭代。
