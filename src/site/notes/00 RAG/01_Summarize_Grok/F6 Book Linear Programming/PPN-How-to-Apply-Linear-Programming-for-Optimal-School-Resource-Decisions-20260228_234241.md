# PPN- How to Apply Linear Programming for Optimal School Resource Decisions? 如何用線性規劃優化學校資源決策？


## 1. 核心洞見
- **可行域定義領導邊界**：線性不等式系統如學校預算、師時、人力限制，形成feasible region。領導者須先圖形化畫出陰影區域，辨識真正可執行的策略空間，避免盲目衝動。助理校長以此視角審視校政，轉化混亂約束為清晰決策地圖，提升資源效率30%以上。

- **極值總在頂點發生**：線性函數P在可行域頂點達max/min，喻領導目標（如升學率、師生滿意）必於關鍵交點實現。非邊緣方案難極致，教我們聚焦轉折點如跨科整合或預算重分配。學校應用：模擬不同情景，快速鎖定高回報決策。

- **圖形化思維加速洞察**：從陰影圖寫不等式，反之亦然，訓練視覺直覺解決複雜系統。教育領導借鏡，將校務數據視覺化為坐標圖，瞬間見資源瓶頸。長期視野：培養教師此技能，建校本數據文化。

- **約束轉化為機會**：不等式如x≤-1、y≥2代表硬邊界，但陰影區內仍有無限可能。領導哲思：視政策限制為優化起點，非阻礙。香港中學實踐：DSE 3A/3B題目啟發，將通識壓力轉為跨學科項目max學生潛能。

- **線性規劃應用日常**：從word problems到public exam，強調實際如生產、分配。校長視角：預算max教學產出，師生比min流失率。哲思深度：非零和遊戲，優化即win-win，提升校園韌性。

- **頂點決策的領導藝術**：點A/B座標精準計算，如(5,1)達17，教我們量化評估方案。助理校長應用：模擬師資調配，避平均主義陷阱，直擊高影響點。視野升級：從直覺到科學領導，複利校績。

- **系統整合跨章洞見**：3C極值連3D應用，喻學校非孤島，決策須holistic。open-ended及integrated questions提醒：真領導考驗綜合視野。香港校園：建模DSE數據，預測並優化升學路徑。

## 2. 行動建議
### 給自己（個人思維層面）
- 每日默想：將一校務問題轉為不等式系統，問「我的feasible region在哪？頂點方案何也？」升級決策從gut feel到analytical。

- 內化頂點哲學：記住「極值在邊界」，拒絕舒適中庸，訓練挑戰約束思維，如預算緊絀時尋交點機會。

- 建個人領導坐標：x軸資源、y軸影響，定位日常決策，養成視覺化習慣，提升直覺精度。

### 給日常實踐（立即可執行的行動）
- **今日起**：選一科組，列3不等式（師時≤40、人數≥30、預算<50k），畫陰影區，計P=升學貢獻max點，1小時內出報告。

- **1-3天**：用Excel模擬校預算線性規劃，輸入DSE-like數據，找頂點方案，與主任分享，測量潛在節省10%。

- **即刻應用**：課堂示範3A題，問學生「若你是校長，如何max快樂min壓力？」記錄洞見，融入晨會分享。

- **當週追蹤**：每日評一決策是否在feasible region內，記錄頂點收益，養成量化反思。

### 給長期系統（建立習慣或流程）
- **建校本LP框架**：每月工餘研習會，教職員學畫不等式圖，建template庫（Obsidian MD），追蹤跨年優化，如師生比調整。

- **複利機制**：設KPI dashboard，視覺化可行域變化，季度審視頂點移位，連結校績獎勵，文化化數據領導。

- **持續培訓流程**：DSE 3C為藍本，設計校內模擬賽，教師輪流當「校長」優化虛擬情景，年複利提升決策效能20%。

- **遺產系統**：記錄每學年頂點案例至Obsidian vault，供繼任參考，建「領導規劃手冊」，確保學校韌性世代傳承。
