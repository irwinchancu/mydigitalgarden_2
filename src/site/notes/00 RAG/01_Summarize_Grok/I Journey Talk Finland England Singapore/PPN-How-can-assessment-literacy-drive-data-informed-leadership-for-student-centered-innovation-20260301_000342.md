# PPN- How can assessment literacy drive data-informed leadership for student-centered innovation? 評估素養如何驅動數據導向領導，實現學生中心創新？



## 1. 核心洞見
- **評估從測量轉向賦能**：傳統 summative assessment 只驗證學習成果，但 AfL/AaL 強調 formative feedback，讓學生主動反思，教師數據分析弱點。Backward Design 先定目標、再設計證據、最後規劃活動，避免盲目教學。香港學校應用此框架，可將評估數據轉化為即時教學調整，提升學生 metacognition。

- **數據分析解鎖學習痛點**：多位教師分享分析試卷分題成績、個人報告、Ctl rubrics（行為、努力、準時、韌性），找出學生難點並跟進。非僅問責，而是診斷根本，如疫情下虛擬數據收集。領導視野：建校本數據儀表板，跨科共享，實現精準教學。

- **學生主導評估培養自主**：EAT Framework 及 self/peer assessment 資源包，讓學生建 rubrics、反思進程，提升 assessment literacy。S4 寫作用 AI 回饋、PEEL 結構，學生自信上升 16%。哲思：從被動接受到 active learner，學校文化轉向 trust-based autonomy，如芬蘭 Less is More。

- **跨學科整合放大影響**：STEM/STEAM 項目融合科學、工程、數學、藝術，用 Design Thinking、SCAMPER，解決真實問題如智能溫室。現象學習（phenomenon-based）打破學科壁壘，培養 6Cs（collaboration, critical thinking 等）。領導應用：初中跨科專題，教師 PLC 協作，複利學生共通能力。

- **創業教育注入內在動機**：芬蘭內在 EE（entrepreneurial skills）強調 creativity、risk-taking，非商業。香港項目如 Happiness 商務、Youth Flea Market，讓學生 ideation、pitch，提升 resilience。視野：將 EE 嵌入 curriculum，轉化失敗為學習，建校園 innovation hub。

- **教師專業社群是關鍵槓桿**：i-Journey 啟發建 Task Force、lesson study、sharing sessions，提升教師 literacy。疫情下轉線上 survey、virtual trips，仍見成果。哲思：領導者非指令，而是 facilitator，透過 PLC 散播 best practices，持續 PD。

- **系統評估需多維追蹤**：pre/post-test、survey、interview 驗證成效，如 Flourishing Life Programme 測 well-being。疫情挑戰下，強調 follow-up 及迭代。洞見：建 balanced scorecard，融合量化（如分數提升）與質性（如動機），確保可持續。

## 2. 行動建議
### 給自己（個人思維層面）
從 assessment as judgment 轉向 growth mindset，每週反思一堂課數據：學生哪裡卡住？我 feedback 是否 actionable？採用 EAT Framework 審視自身教學，問「學生如何用我的評估前進？」每日 journaling 記錄一 insight，培養 data-informed intuition，像 McKinsey 般策略思維。記住，領導始於自省：你是否 trust 學生 autonomy？

### 給日常實踐（立即可執行的行動）
1-3 天內：選一班試行 Backward Design，重寫下週 lesson plan，明確 LOs 及 evidence。分享 AfL 策略於 panel meeting，如 Ctl rubrics 評行為/努力。1-7 天：建簡單 Excel dashboard 分析最近 quiz 分題，發個人報告給 10 名學生，追蹤弱點並給 targeted feedback。試 peer assessment 一堂課，用 rubrics 及 verbal comments，觀察學生互動。

### 給長期系統（建立習慣或流程）
建 Assessment Literacy Task Force，每月 PLC 審數據、迭代 rubrics，跨科共享如 STEM Quartet。推校本 AfL policy：每科 20% formative tasks，追蹤 via Google Form/survey，每季 report 給校董。設 student-led innovation day，融合 EE/跨科，評以 6Cs portfolio。年度 PD cycle：i-Journey-style sharing，建 digital repository。複利設計：每月 KPI（如學生 self-assess 率 >80%），獎勵優秀 panel，文化轉向 collaborative excellence。
