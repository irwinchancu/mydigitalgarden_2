# PPN- How can mental models build antifragile leadership in schools_ 心智模型如何打造反脆弱校領導？


## 1. 核心洞見
- **避免聯集謬誤與過度配適**：領導者常將複雜事件過度解釋為罕見巧合，如將學生個案歸因於單一因素，而忽略簡單解釋。這在學校政策制定中易導致資源浪費；應用奧坎簡化論（Occam's Razor），優先簡單模型，提升決策效率，讓學校更反脆弱。

- **運用最尊重的解釋（MRI）與無知之幕**：面對師生衝突，先假設對方善意，建立信任橋梁；無知之幕迫使從弱勢視角思考政策，如遠距學習對單親家庭影響。這哲思深化同理心，轉化為公正領導視野，避免公正世界假說（Just World Hypothesis）譴責受害者。

- **灰色思維對抗驗證性偏誤**：擁抱矛盾觀點，如同時肯定傳統教學與創新科技價值；魔鬼代言人（Devil's Advocate）強迫團隊辯論，提升決策客觀性。在學校轉型中，這避免極端，培養耐心，接近真相。

- **五個為什麼與事後檢討**：挑戰者號災難揭示組織失靈根源；學校領導須追根究柢，如學生輟學背後是溝通失敗，而非表面懶惰。定期postmortem建構學習文化，讓失敗成反脆弱燃料。

- **技術負債與完美是敵人**：短期修補如管理負債（management debt），長期累積危機；區分可逆（reversible）與不可逆決定，快速迭代學校流程。貝佐斯單向門（one-way door）模型，釋放創造力，避免拖延。

- **折現偏差與預設值效應**：當前偏差致拖延，領導須預設長期承諾，如行事曆鎖定深度工作。帕金森定律提醒工作膨脹時間；侯世達定律（Hofstadter's Law）警示估時偏差，建校務複利系統。

- **數據偏差警覺**：軼事證據（anecdotal evidence）、選擇偏差（selection bias）扭曲校務評估，如只看優等生忽略落後者。中央極限定理（Central Limit Theorem）助理解樣本代表性，精準政策數據驅動。

- **文化高低情境與團隊績效**：香港學校多元文化，須明確規範低情境溝通，避免誤解。新人適應期坦誠文化fit，高槓桿領導強化「主管不在時的文化」。

## 2. 行動建議
### 給自己（個人思維層面）
每日反思決策，用MRI解讀師生行為，問「最尊重解釋是什麼？」；練習無知之幕，模擬弱勢角色審視政策，破除公正世界假說。擁抱灰色思維，記錄一對矛盾觀點並整合，提升反脆弱心態。追問五個為什麼剖析個人失誤，轉化為領導智慧。

### 給日常實踐（立即可執行的行動）
1-3天內：列利弊清單加權分數（cost-benefit analysis），評估一項校務決定，如課程調整，避免過度配適。明日起行事曆預設一小時深度工作，克服折現偏差。會議指定魔鬼代言人，挑戰團隊偏誤，持續一週。

### 給長期系統（建立習慣或流程）
建每月事後檢討會議，用五個為什麼分析校務事件，追蹤改善KPI，形成反脆弱循環。設計文化手冊，定義高低情境規範，新人入職必讀並面談fit。導入預設承諾機制，如401(k)-式教師PD計畫自動註冊，結合帕金森定律，每季審核時間分配，複利提升校效能。
