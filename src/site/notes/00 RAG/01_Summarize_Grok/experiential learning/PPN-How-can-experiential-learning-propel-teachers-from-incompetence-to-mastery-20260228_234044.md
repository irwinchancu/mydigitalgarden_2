# PPN- How can experiential learning propel teachers from incompetence to mastery? 體驗學習如何推動教師由無能至熟練？


## 1. 核心洞見
- **四階段學習模型揭示領導視野**：從unconscious incompetence到unconscious competence，每位教師皆需自省當前階段。助理校長可視此為診斷工具，引導教師意識缺陷並尋求coaching，加速掌握。哲思上，這強調教育領導非強迫，而是喚醒內在動機，建構學校持續成長文化。

- **體驗學習超越hands-on，強調反思**：學習金字塔顯示「做」保留90%，但無反思僅止於表面。教育領導須視反思為核心，將經驗轉化洞見，避免「做了卻未學到」。此哲思提升學校教學深度，培養學生終身學習力。

- **八原則建構高質體驗框架**：從Intention到Acknowledgement，涵蓋全過程確保真實性與持續改善。領導者以此審視學校活動，如PBL，確保不流於形式。哲思在於：體驗非事件，而是系統，助香港中學應對高壓考試文化轉向深度學習。

- **PBL過程激活協作問題解決**：討論、腦storm、反思循環，教師轉facilitator角色，非權威資訊提供者。學生獲深度認識、自我導向與批判思維。領導視野：此模式重塑課堂動態，減低教師負荷，提升全校效能。

- **概念澄清問題深化哲思對話**：如「What exactly does this mean?」，迫使學習者自省本質。助理校長應用於會議或觀課，轉化經驗為智慧。哲思核心：教育非傳遞知識，而是喚醒潛能，建構反思型學校文化。

- **PBL優勢複利於學校生態**：學生增強動機與横向思考，教師受益於和諧氛圍。領導須統合Dewey等先賢理念，視PBL為轉型引擎。長期價值：香港中學由此從應試邁向創新，培養未來領袖。

- **評估與認可確保可持續**：監測非懲罰，而是refine目標；celebration提供closure。哲思上，這體現人文領導，強化師生歸屬感，避免燒盡。

## 2. 行動建議
### 給自己（個人思維層面）
- 自評四階段：每日反思教學技能，如「我對PBL是否conscious incompetent？」，轉變心態為lifelong learner。

- 內化八原則框架：視每校務為體驗，問「有無reflection？」，升級決策為系統思維。

- 擁抱facilitator角色：放棄「全知」假象，練習概念澄清問題，培養深度傾聽，提升領導自信。

### 給日常實踐（立即可執行的行動）
- 明日觀課應用：選一節課，記錄教師階段並即時coaching，如示範PBL討論，1週內覆蓋5位教師。

- 啟動微PBL：1-3天內，召集團組腦storm一校務問題（如學生動機），執行行動計劃並當日反思，追蹤保留率提升。

- 每日反思日誌：課後問「What did we do? What learned?」，分享於教員群組，7天內養成習慣。

### 給長期系統（建立習慣或流程）
- 建校級PBL追蹤表：每月評估八原則應用，KPI如「反思次數>80%」，複利教師熟練。

- 教師成長圈：每週facilitator培訓，輪流主持概念澄清討論，連結績效評核，持續3個月見文化轉變。

- 慶祝機制：季度「體驗影響分享會」，記錄成就於Obsidian資料庫，強化認可循環，建構可持續學校領導系統。
