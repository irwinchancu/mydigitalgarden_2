# 動他的家長或兒童模式

### 1. 教師應以「Adult（成人 / 正事模式）」為主要狀態

- **適用處境**：
    - 講解知識、帶領討論、解決問題、規劃活動、給予客觀回饋。
    - 學生出現學習困難、需要理性分析原因時。
    - 課堂常規時間（大部分教學時段）。
- **為何要用**：這是讓學生進入「正事模式」的最有效方式。Adult 對 Adult 的溝通最平穩、不帶情緒，學生容易跟隨。
- **實踐方式**：用事實、數據、問題導向語言，例如：「我們來看看這個問題的數據…你覺得下一步應該怎麼做？」

### 2. 「Parent（家長模式）」的適用處境

- **適用處境**：
    - 需要設定界限、執行紀律、保護學生安全時（例如學生互相欺負、違反班規、危險行為）。
    - 給予明確指令或結構化指導（例如考試規則、作業要求、緊急情況）。
    - 學生情緒崩潰、需要被「照顧」或「保護」時（Nurturing Parent）。
- **為何要用**：家長模式能迅速建立秩序與安全感，但只能短暫使用，且必須是「正面家長」（Nurturing / Guiding Parent），避免變成批評型（Critical Parent）。
- **注意事項**：用完後要盡快切回 Adult，否則學生會長期處於「兒童狀態」依賴或反抗。

### 3. 「Child（兒童模式）」的適用處境

- **適用處境**：
    - 建立關係、破冰、激發創意、減輕壓力時（例如遊戲化學習、團隊活動、慶祝小成就）。
    - 學生非常緊張或抗拒學習，需要先「玩一下」來鬆弛氣氛。
    - 適度展現幽默、分享個人趣事，拉近距離（Free Child）。
- **為何要用**：能增加親切感與動機，但只能「少量、刻意」使用。
- **注意事項**：教師絕不能長時間陷入 Child 模式（例如跟學生一起鬧、情緒化抱怨），否則學生無法進入正事模式。這是很多新手老師最容易犯的錯誤。

### 實務切換原則（優秀老師的日常做法）

|處境類型|優先模式|次要/輔助模式|切換時機提示|
|---|---|---|---|
|常規教學、解題|Adult|-|整堂課 80% 以上|
|學生違規、需紀律|Parent|Adult|發現問題 → 立即 Parent → 馬上回 Adult|
|破冰、團隊活動|Adult|Child|先 Adult 說明規則 → 短暫 Child 帶動 → 回 Adult 總結|
|學生情緒低落|Parent (Nurturing)|Adult|先安撫 → 快速轉 Adult 解決問題|
|考試前壓力大|Adult|Child (少量)|Adult 主導 + 短暫幽默減壓|

### 總結與提醒

- **核心原則**：教師永遠以 Adult 為「預設模式」，只在必要時短暫切換 Parent 或 Child。切換後必須有意識地立刻回到 Adult，這就是「事情沒有你想的那麼簡單」的原因。
- 優秀老師每天都在練習「自我覺察」：上課前問自己「我現在是哪種狀態？」，課後反思「今天有沒有不小心用錯模式？」。
- 如果您能長期維持 Adult 為主，學生自然會跟著進入正事模式，課堂會變得穩定而高效。



因為要是老師切換至家長或兒童模式，其他學生會觀察老師的反應，並且輕而易舉地跟著切換至兒童模式。正因如此，同學才覺得我很搞笑，或至少比他們在正事模式時來得好笑。一旦某位老師開始咆哮，以非正事模式衝著我說教，你可以聽見其他學生啟動模式開關的聲音。噠噠噠，切換成兒童模式。一旦教室裡的學生常常進入兒童模式，要改變他們就難了。事實上在踏進你的教室前，他們就已經是這種心態。為了讓學生進入正事模式，你自己就得先處於正事模式，若你是家長或兒童模式，就不可能讓他們轉為正事模式，事情沒有你想的那麼簡單。優秀老師知道這件事，也天天都在實踐。

父母的角色與教師略有不同：您天生會較常處於「Parent（家長模式）」，這是正常的養育本能。但若想讓孩子真正進入「正事模式」（即理性、負責、自主學習與解決問題的成人狀態），您仍必須以「Adult（成人 / 正事模式）」為主要狀態，並有意識地控制模式切換。這正是「事情沒有你想的那麼簡單」的原因——許多父母以為「我就是管教他」就能讓孩子變乖，卻忽略自己若長期停留在家長模式，孩子便難以培養獨立思考與自我管理能力。

### 1. 父母應以「Adult（成人 / 正事模式）」為主要狀態

- **適用處境**：
    - 日常溝通、討論學習、解決問題、規劃家庭事務、給予生活建議。
    - 孩子遇到挫折、需要分析原因或做出選擇時。
    - 親子對話的大部分時間（建議佔 70% 以上）。
- **為何要用**：Adult 對 Adult 的互動最能培養孩子的責任感與理性思考，讓孩子學會「自己思考、自己負責」。
- **實踐方式**：用事實、問題導向語言，例如：「這件事發生了，你覺得原因可能是什麼？下一步你可以怎麼做？」

### 2. 「Parent（家長模式）」的適用處境

- **適用處境**：
    - 需要設定界限、執行家規、安全保護時（例如孩子做危險事、違反作息、傷害他人）。
    - 給予明確指令或結構化指導（例如作業時間表、家庭責任分工）。
    - 孩子年紀很小、情緒崩潰或生病時（以 Nurturing Parent 溫柔照顧為主）。
- **為何要用**：這是父母的天然職責，能提供安全感與方向，但只能「短暫、必要」使用，且盡量用正面引導型（Nurturing / Guiding Parent），避免變成批評或控制型（Critical Parent）。
- **注意事項**：用完後必須立刻切回 Adult，否則孩子會長期依賴或產生反抗心理。

### 3. 「Child（兒童模式）」的適用處境

- **適用處境**：
    - 建立親密關係、減輕壓力、激發快樂時（例如一起玩遊戲、慶祝成就、分享趣事）。
    - 孩子非常緊張或抗拒時，先用短暫的輕鬆互動破冰。
- **為何要用**：能增進親子感情與動機，但必須「極少量、刻意」使用。
- **注意事項**：父母絕不能長時間陷入 Child 模式（例如跟孩子一起發脾氣、互相鬧、抱怨生活），否則孩子無法學會正事模式。這是很多父母最容易忽略的陷阱。

### 父母常見處境模式搭配表

|處境類型|優先模式|次要/輔助模式|切換時機提示|
|---|---|---|---|
|日常親子對話、討論學習|Adult|-|整天大部分時間|
|孩子違規、需訂界限|Parent|Adult|發現問題 → Parent 說明後 → 立刻回 Adult|
|孩子情緒低落或害怕|Parent (Nurturing)|Adult|先溫柔安撫 → 快速轉 Adult 一起解決|
|家庭遊戲、放鬆時間|Adult|Child（短暫）|先 Adult 訂規則 → 短暫玩樂 → 回 Adult 總結|
|教導生活技能、責任|Adult|Parent（少量）|Adult 主導，讓孩子自己動手|

### 父母實踐核心原則

- **預設模式永遠是 Adult**：即使您是父母，也要先把自己調整到理性、平穩的成人狀態，才能帶領孩子進入正事模式。
- **自我覺察是關鍵**：每天問自己三次：「我現在是哪種狀態？對孩子有幫助嗎？」晚上睡前簡單反思今天有沒有不小心用錯模式。
- **長期目標**：讓孩子逐漸學會用自己的 Adult 狀態與您互動，這才是真正的教養成功。
- **常見錯誤**：很多父母以為「我生氣管教就是家長模式，這樣孩子才聽話」，但其實過度使用只會讓孩子學會「表面順從、內心反抗」或「完全依賴」。

總結來說，作為父母，您比老師更有優勢——因為您與孩子的關係更深、更長久。只要堅持以 Adult 為主、Parent 為輔、Child 為點綴，孩子自然會跟隨您進入正事模式，變得更有責任感、更獨立，也更懂得解決問題。

若您能告訴我孩子的年齡（例如幼兒園、小學、國中）、具體困擾情境（例如不肯寫功課、叛逆頂嘴、情緒管理等），我可以給您更精準的日常對話範例與模式切換腳本。歡迎繼續分享，我會盡力提供專業建議。

以下為針對中一學生（約12–13歲）的「艱難對話範本」。我選取了香港中一常見的5個高衝突情境，每個範本均以「交易分析理論」為基礎，示範如何以 **Adult（成人 / 正事模式）** 為主導，必要時短暫使用 Nurturing Parent（溫柔引導型家長模式），並迅速切回 Adult。

每一個範本都包含：

- 情境簡述
- 錯誤示範（常見家長模式過度使用）
- 正確對話腳本（帶模式標註）
- 為何有效

### 範本1：孩子不肯做功課（最常見）

**情境**：晚上8點，孩子躺在床上滑手機，功課只做了三分之一。

**錯誤示範**（過度 Critical Parent） 家長：「你怎麼又不做功課？整天只顧玩手機！你這樣將來怎麼讀中二、中三？真係廢！」

**正確腳本** 家長（Adult）： 「我看到你功課只做了三分之一，現在已經8點了。我們一起看看，哪幾科最急？」 孩子：「唔想做，好攰。」 家長（短暫 Nurturing Parent）： 「我明白今日學校已經上足6堂，確實好攰。先喝杯水，休息5分鐘，之後我們一起把最難的那一科做完，好嗎？」 （5分鐘後，切回 Adult） 家長：「好，現在開始。你負責做數學第3–5題，我坐在旁邊，有問題隨時問我。做完這幾題我們就完成今日目標。」

**為何有效**：用事實開頭（Adult），先共情再給結構，孩子感覺被尊重，不會即時反抗。

### 範本2：孩子沉迷手機/遊戲

**情境**：週末孩子連續玩遊戲超過4小時，飯也不好好吃。

**錯誤示範** 家長：「把電話交出來！每日限1小時你都唔聽，係咪想我扔咗佢？」

**正確腳本** 家長（Adult）： 「我留意到你今日已經玩了超過4小時遊戲。我們之前講過，每日最多2小時。你覺得呢個安排點解會幫到你？」 孩子：「我玩得開心啫！」 家長（Adult）： 「開心係好重要，但長期玩超過2小時會影響眼睛同埋睡眠。你可以選擇：現在關機，之後我們一起計劃下星期點樣分配時間；或者我幫你設定計時器，時間到就自動鎖機。你選邊個？」 （孩子選了計時器） 家長（Nurturing Parent + Adult）： 「好選擇。我相信你有能力自己管理，爸爸/媽媽會支持你。」

**為何有效**：把選擇權交給孩子（Adult對Adult），孩子更有責任感，不會覺得被「搶」手機。

### 範本3：孩子成績下滑 + 頂嘴

**情境**：中一第一次測驗，數學只得55分，家長問原因時孩子即時頂撞。

**錯誤示範** 家長：「55分？你中一就咁差？以後點讀書？真係氣死我！」

**正確腳本** 家長（Adult）： 「今次數學測驗55分，我睇到你錯咗幾題應用題。我們一起分析下原因，好嗎？」 孩子：「關我咩事？老師教得差啫！」 家長（Adult，保持平穩）： 「老師教得點樣係另一件事，但你自己有冇每日溫習10分鐘呢？我想聽聽你嘅想法。」 （孩子沉默或繼續推卸） 家長（Adult）： 「好，我明白你覺得唔係自己問題。咁樣吧，我們訂一個為期兩星期的計劃：每日放學後先做20分鐘數學練習，我陪你檢查。如果你做到，我會同你一齊慶祝；如果做唔到，我們再調整計劃。你同意嗎？」

**為何有效**：不爭辯「誰的錯」，直接聚焦「下一步怎麼做」，孩子慢慢學會承擔責任。

### 範本4：孩子要求更多自由（晚上想出去玩）

**情境**：中一孩子說「同學都成日出去，我點解唔可以？」

**錯誤示範** 家長：「唔准！中一就想夜街？返屋企讀書先！」

**正確腳本** 家長（Adult）： 「我明白你想同同學出去玩，這代表你開始想有更多自由。我們可以談談。」 孩子：「其他同學父母都准！」 家長（Adult）： 「每家情況唔同。我擔心的是安全同時間管理。你可以提出一個計劃：幾點出、幾點返、同邊幾個同學、去邊度？我聽完再決定可唔可以。」 （孩子提出計劃） 家長（Adult）： 「你嘅計劃聽落好有條理。咁樣吧，先試一次星期六下午3點到6點，返到屋企要即刻同我報平安。如果呢次做得好，下次可以再加長。你覺得呢個安排公平嗎？」

**為何有效**：承認孩子成長需求（Adult），用「試行計劃」代替一刀切，讓孩子練習負責任。

### 範本5：孩子情緒崩潰（考試前焦慮大哭）

**情境**：孩子因為測驗壓力大哭，說「我讀唔到，唔想讀書了」。

**錯誤示範** 家長：「哭咩哭？其他同學都讀到，你點解咁弱？」

**正確腳本** 家長（先 Nurturing Parent）： 「我看到你好擔心，過嚟，我抱抱你。」（給予肢體安慰，30秒） （切回 Adult） 家長：「現在感覺好啲了嗎？可以同我講講，你最擔心邊一科？」 孩子：「全部都擔心…」 家長（Adult）： 「好，我們一科一科嚟。先列低你覺得最難嘅3個課題，之後我幫你拆解成細步，每日只做一步。你覺得呢個方法可唔可以減輕壓力？」

**為何有效**：先安撫情緒（必要 Parent），再快速轉 Adult 解決問題，避免孩子長期停留在 Child 狀態。

### 總結使用技巧（中一適用）

- 開口前先深呼吸，確認自己處於 Adult 狀態。
- 每句話以「我看到…」「我們一起…」「你覺得…」開頭，避免「你點解…」「你唔准…」。
- 每次對話結束後，晚上私下問自己：「今日有冇用錯模式？」
- 中一孩子正值「想獨立又未成熟」的階段，坚持3–4星期用 Adult 主導，你會發現孩子頂嘴明顯減少、主動承擔增加。

如果您想針對某一個情境再要更詳細的「連續3天對話腳本」、或提供孩子最近的具體事件（例如他說了什麼頂嘴說話），我可以即時再度身訂做更精準的範本。請告訴我您最想先處理哪一項，我會繼續協助。

針對您中一（12–13歲）的孩子，這段建議的核心技巧（沉穩重複「沒寫完，什麼都不必說」，不爭辯、不說理）在香港很多中一家庭中已被證實有效，尤其適合處理「先玩再寫」這種經典的本末倒置測試。我根據交易分析理論（Transactional Analysis）及多年實務經驗，為您分析如下：

### 為什麼這個方法有效？

1. **中一孩子正值「底線測試期」** 他們開始追求自主，卻尚未具備完整責任感，會不斷用「先玩再寫」來試探父母是否會妥協。重複簡單、堅定的句子，正好用**簡短有力的 Parent 模式**（Guiding Parent／引導型家長模式）劃清界限，讓孩子清楚知道「這件事沒有談判空間」。
2. **避免情緒升級** 如建議所說，在孩子情緒當下說理，只會進入 Child vs Parent 的交叉交易，雙方愈吵愈僵。沉穩重複同一句話，等於保持**Adult 平穩頻道**，不讓孩子「得逞」把你拉進情緒戰。
3. **建立長期習慣** 堅持3–7天後，大部分中一孩子會發現「試探無效」，自然減少挑戰。很多家長回報：第1週可能要重複5–8次，第2週減至1–2次，第3週孩子已自動先寫完再玩。

### 潛在風險與優化建議（最重要）

純粹只用「沒寫完，什麼都不必說」屬於**純 Parent 模式**，短期有效，但若長期只用，可能令孩子：

- 表面順從（怕你），卻內心積壓反抗；
- 無法學會「為什麼要先寫」——即未能培養自己的 Adult 狀態。

**最佳做法**：把這個技巧當作「開場白」，之後迅速切回 Adult 模式，讓孩子慢慢內化責任。

### 優化版完整對話腳本（中一適用）

**情境**：晚上7點，孩子說「我先玩一陣再寫啦～」

**您的回應（沉穩、眼神專注、語調往下壓）**： 

家長（Adult + 堅定 Parent）： 「沒寫完，什麼都不必說。」
