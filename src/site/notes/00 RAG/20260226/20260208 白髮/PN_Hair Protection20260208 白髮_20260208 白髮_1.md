# 全齡逆轉白髮聖經

在日照強烈的大熱天底下進行休閒活動或是水上活動時，建議選擇「SPF 30」或是「PA++++」的產品。SPF數值較高的產品對於頭皮比較刺激，所以要抹在頭皮的話，最好選擇SPF數值低於30的產品。

# 膻中穴

位於左右兩側乳頭連線處的中心點，也就是胸口的正中央之處。

膻中穴與心理狀態息息相關，可幫助我們消除不安的情緒。

以「按壓六秒，休息六秒」的節奏按壓三次，力道控制在覺得舒服的範圍即可。
