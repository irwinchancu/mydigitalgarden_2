## 重點工作一：接觸當事人、評估和協助處理危機

首要的工作是接觸當事人，評估他是否面對危機及立即給予援助。當中的重點是：

- 接觸有需要的青少年  
- 選擇一個令雙方都感到舒服且合適的時間和地點  
- 若當事人沒有主動表達感受，你應該主動關心他，和他談談  
- 尊重當事人的私隱和保守秘密  

而青少年當事人出現的危機包括：

- 傷害自己，例如：試圖自殺、物質濫用、自我傷害行為  
- 當事人感到嚴重困擾，例如：驚恐性發作、面對創傷事故、出現重性精神病徵狀  
- 出現滋擾他人行為，例如：有暴力傾向、脫離現實  

若經初步評估當事人並非處於危機，可嘗試鼓勵當事人說出現時的感受及何時開始出現該感受，並進入重點工作二。

# 嘗試不同的環境並選擇其中最有利於溝通的

有些成年人認為在一些活動中比較容易與青少年傾談。父母認為單獨與子女在外面進餐是一個與他們傾談的良機。另一些人則認為在坐車的途中、在做家務或一起放狗的過程中，都是與青少年子女傾談的好場合。同樣地，青年工作者、教師或社工來說，運動環境可能較為適合。一些涉及較少眼神接觸的，又或者一些有時限的活動，可以令青少年感到較少壓迫感及較易表達。絕對適合與青少年溝通的場合並不存在。告訴當事人你願意提供協助並與他一起找尋最適合大家溝通的場地。

# 不要與當事人比較你在他那個年紀時的經驗

成年人經常錯誤地認為現今的青少年生活過得比較容易。不要忘記在你的年代，你父母亦有同樣的看法。有些說話如「如果我在你的年紀時，有你現時那麼多的機會，我便會…」是沒有意義的，世界不斷在變，新的機會同時意味著新的挑戰。

# 不要輕視青少年的感受

精神健康問題可以在任何年紀出現。不要因為他們年紀小而看輕他們的生活經歷，繼而懷疑他們是否真的抑鬱或焦慮。

# 不要要求青少年為他的行為提供合理解釋

青少年通常沒有考慮清楚便行動，到事後才發現他們犯了錯誤。要求青少年去解釋為何明知事情是不當仍要去做是沒有意義的，反而，提點他們避免再犯上述錯誤是較有效的。要他們去解釋或者問「為甚麼」，會使青少年推向築起防衛機制，不願意坦誠表達。

# 注意你的身體語言

不論你與甚麼人傾談，這都是重要的。這對青少年來說尤其重要。成年人的身體語言可能已不自覺地表達了「我是專家」的訊息。雙臂交叉放在胸前、雙手叉腰及其他權威性或防衛性姿態使對話難以有效進行。如果青少年表現得放鬆及開放，你可嘗試表現相配合的姿態。如果青少年表現得防衛，你便盡量表達開放的姿態：如神情輕鬆、雙手放下，坐在當事人身旁，以平靜、溫和的語調展開對話。

# 給予正面回應

青少年經常被父母、老師或傳媒指出他們的不是，因此你給予的正面回應能讓當事人較願意與你溝通，就算簡單句子如「你願意與我談及這問題真是很好，你已表現成熟…」亦有助溝通。


# 表達關注及專注聆聽

與青少年對談時，多用「我」而非「你」，例如：「我為你感到擔心/憤怒/挫敗..」而不要說：「你令我感到擔心/憤怒/挫敗…」詢問當事人有關自我傷害的行為，但不要給予壓力強迫他對話。當他說出自己的經驗及情況後，你可以反映歸納他的形容。

# 給予支持及安慰

對於青少年的感受，表達同理心。認同當事人的情緒，並解釋這些情緒是合理及正常的。

讓當事人知道他並不孤獨，你願意支持他。與當事人一同尋找解決方案，例如，了解發生了什麼事，討論可行的解決方法等。

安慰當事人，他是可以找到資源及幫助的。告訴他，你願意提供協助，並讓他知道你可以提供的協助方法。

不要隨意答應他保守秘密。如果你為保安全，而要告訴第三者有關當事人自我傷害的行為，你應當先通知當事人。不要在未得當事人允許下，在他背後討論這事。

# 與有自我傷害行為的當事人對話時

- 不要小看當事人的感受及面對的困難  
- 不要說些不認真看待當事人痛楚的話，例如：「其實你的人生非常美滿。」或「事情並不是那麼差。」  
- 不要嘗試為當事人解決他的問題  
- 不要在未得同意前，觸碰當事人（例如擁抱或攙手）  
- 不要在當事人面前形容他為：自殘者、自傷者  
- 不要判斷當事人的行為是為了吸引別人的注意力  
- 不要讓當事人因為自我傷害，影響到別人而感受到罪疚感  
- 不要在當事人主動提出之前，與他定立目標或契約，例如：如果你答應我一個星期内不傷害自己，你就做得好！  
- 不要用威脅的方式來停止自我傷害行為，例如：如果你繼續傷害自己，你就要搬出去！  
- 不要自行為當事人處方藥物或酒精

### 如何判斷情況是否嚴重

所有自殺念頭都必須認真地對待及給予行動回應。不要認定當事人的想法是「吸引注意力」或「只是想人幫助」，應當因應自殺危機訊號而決定行動的緊急性。

要確定當事人是否有安全問題，可以問有自殺念頭的當事人以下問題：

- 有沒有自殺計劃  
- 打算如何自殺，例如直接問當事人，他打算如何及在哪裡自殺  
- 計劃何時自殺  
- 有沒有作出行動，實行自殺計劃  
- 有沒有服用藥物或酒精，藥物或酒精中毒會增加自殺危機  
- 過去是否嘗試自殺  

如果意圖自殺的當事人表示，他聽到聲音，可以問他那些聲音說了些什麼。如果那些聲音與他的自殺念頭有關，則是非常重要的事。

了解當事人有什麼支援資源，將非常有用。你可以問當事人：

- 有否告訴任何人他的感受  
- 他的學習、社交或家庭生活有沒有轉變  
- 有沒有接受精神健康問題的治療或服用精神科藥物  

越詳盡的計劃，例如已有詳細計畫、執行時間及意圖，意味著越高的風險。可是，你也要謹記，沒有計劃也不足以確定當事人是安全的。

### 如何協助

#### 如何與有自殺傾向的青少年對話

你應當表現真誠，而並非只是說「正確的事」。你可以對當事人表達支持及了解，專注地聆聽。自殺念頭經常是一種求助的表達，當事人在絕望時會視自殺為逃避困境和擺脫煩惱的最後方法。
