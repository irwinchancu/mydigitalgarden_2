# 剛剛好的自由

1. **青春期情緒風暴是多因素綜合作用，而非單純「叛逆」** 生理因素包括荷爾蒙劇烈變化、大腦杏仁核過度活躍、睡眠不足及血糖偏低；心理因素則涉及自我認同建構與價值感不足；外部因素有過度管控、家庭或同儕壓力。 **班級應用**：當學生在課堂突然出現無名火或沉默抗拒時，先避免立即對抗。給予短暫「冷靜空間」（例如讓他到教室後方或走廊片刻），待情緒稍緩後，以中性語句標籤化：「我看到你現在看起來很焦慮／生氣……需要我幫忙嗎？」這能有效降低衝突，並示範健康的情緒處理模式。

2. **情緒沒有對錯，但表達方式需要引導與練習** 所有情緒均屬自然，先允許其存在，再引導學生「看見」並適切表達（言語、運動、寫作等）。 **班級應用**：可在班上設立簡易「情緒角落」或定期教授2-3分鐘的呼吸／正念練習。對BAND 2學生而言，這類工具特別有助於應對考試前焦慮或小組合作時的情緒衝突，長遠提升學習專注力與同儕關係。

3. **清晰的界線是尊重，更是雙方保護** 書中區分五大界線（生理、物質、心理、情緒、性），強調過度介入（如不尊重隱私或過度關心私人事務）會破壞信任。 **班級應用**：在學年初與學生共同訂立班規時，清楚說明「可接受行為」與「後果承擔」。例如：尊重學生手機與私人空間，但明確作業提交與課堂參與屬必須負責範圍。這讓學生在界線內享有自主，同時學習為選擇負責。

4. **退居第二線，給予「剛剛好的自由」與責任** 成人需從微觀管理轉向宏觀引導，讓學生為自身決定承擔後果。 **班級應用**：在專題研習、小組討論或紀律處理中，多讓學生自行規劃時間表與分工，並引導他們反思「如果延誤，後果如何？」BAND 2學生已有相當自主能力，此做法往往能激發責任感與學習動機，減少「被動擺爛」現象。

5. **老師必須先做好自我情緒管理與反思** 書中提供多項自問清單（如「我為什麼出現此情緒？」「我的期望是否合理？」「我希望這段關係如何發展？」），幫助成人避免情緒失控。 **班級應用**：面對頑皮或低動機學生時，先停下來問自己三個問題，能有效防止「玉石俱焚」。建議每週利用10分鐘進行此反思練習，這不僅保護自身心理健康，更成為學生最佳的情緒示範。

6. **正向連結與賞識是長遠穩定的力量** 多肯定優點、避免負面比較、創造日常正向互動（如聊新聞、簡單關懷）。 **班級應用**：在BAND 2學校，可多採用「優勢為本」方法——公開表揚學生努力過程而非單一成績，並鼓勵同儕互賞。這能提升學生自尊、班級歸屬感，同時減輕學業壓力帶來的負面情緒。

**總結** 書中最深刻的一句是：「我們是大人，要幫孩子學習轉換情緒和處理事情，而不是讓自己也情緒失控」。在香港BAND 2學校的高壓環境中，您正扮演關鍵角色——提供一個安全、有界線、同時給予適當自由的學習空間，讓學生練習自律與責任，逐步長成他們自己也喜歡的樣子。這不僅有助學生情緒穩定與學業表現，更能減輕您作為老師的情緒勞動。
