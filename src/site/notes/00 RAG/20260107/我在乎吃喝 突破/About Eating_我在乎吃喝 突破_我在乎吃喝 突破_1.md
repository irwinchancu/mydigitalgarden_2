# 我在乎吃喝


長時間生活在一種追求即時滿足、快捷、速度、方便、就手的消費文化裏面，人自然會愈來愈欠缺耐性和忍耐力。〈馬太福音〉4章2節記載「祂禁食四十晝夜」，禁食是一種重要的屬靈操練，操練人去等候，耶穌則等候了四十晝夜。當祂拒絕了魔鬼的誘惑之後，也許祂要再等下去。禁食就是要操練我們在漫長的時間裏等候的耐性（patience of waiting）；同時，禁食操練我們在時間上延遲滿足慾望的意志和耐力，從而塑造一種節制的德性。因此，面對食慾得到即時滿足的「時間的誘惑」，四十晝夜在飢餓中的忍耐與等候，就是一種「時間的屬靈操練」。



此外，禁食同時提醒我們，基督徒的生命也有另一種「靈糧」作為食物的意涵，也就是祈禱與讀經也在某個角度下能理解為基督徒的飲食；換言之，肉身生命與靈性生命的飲食兩者其實有一種整全平衡的重要性，缺一不可！


而靈性在這文章中，主要是指三方面，
第一：我們以何為樂（滿足感的來源）；
第二：我們是誰（自我形象）；
第三：我們的方向（核心價值）。


現代人有情緒需要，第一時間想到的可能不是朋友，而是食物。有

透過日用品為基層帶來勇氣（Courage）、自信（Confidence）、選擇（Choices）、機會（Chances），然後帶來改變（Change），當中也蘊含尊嚴與希望。神說：「無論何事，你們願意人怎樣待你們，你們也要怎樣待人」（太 7:12）。我相信，神給人希望，也讓人有尊嚴。

### 壓力、寂寞、想放空、尋新鮮，都是都市生活下的合理情緒狀況和需要

我們想以本地有機種植的粟米作為第一道菜，還要生吃不煮。對不少人來說，這算是新奇的食法。港野相信，參加者嚐過鮮甜多汁的粟米，一行一行爬掉數百粒小粟米粒後，有一種自然而生的喜悅，也有完成事情的滿足感。要處理情緒需要，不一定需要薯片、杯麵和新奇食材，一枝有機種植的粟米就有相同之效。結果，在靜默中吃第一道菜時，真的看到那份喜悅從參加者的臉上泛起來
各人非常專注地吃粟米，慢慢的，一粒一粒的，忘記了手機、聽不到環境音樂，嚐到以更好的食物回應自己的需要是怎樣的一回事。

## 為令大家放下有錢就買到食物的扭曲消費觀念，我們預備了來自上水塱原的生態米，配合時令蔬菜，做一碗溫暖身心的糙米粥。

要大家重新感受食物是給我們的恩典，不能夠靠珍饈百味，這只會加強我們的貪婪，強化金錢就是萬能的消費觀念。我們選擇了本地米做食材，因為99.9%的香港人都不知道香港原來仍有人種米；而跟大地連結，感謝生產者和創物主的情懷，必須由認知實況起始。

# 第三道菜：與社區的小店連結

第一道菜看到參加者喜悅的神情，第二道菜則看到參加者驚訝和感恩的表情。接下來的最後一道菜，在專注回應自己的感受，感恩發現食物來源的奇蹟之後，我們再問還有什麼是我們要去連結的？剛巧我們手上有一批本地種植的紅豆，而有位成員想起每天在社區中都會遇到做粄仔糕的陳伯，就忽發奇想，詢問伯伯可否用我們提供的紅豆做糕。陳伯非常樂意幫忙，這就是我們的第三道菜。

「益下家」最根本的概念是「有好嘛益你」，即是人與人之間的分享；意思跟送禮差不多，都是不求回報。喝咖啡的人可以選擇不付款，因為上一位客人早已為他付了；然後，他們也會鼓勵客人付出一杯咖啡的價錢，讓下一位客人也可免費享用咖啡。表面看來，客人還是需要付錢，但付錢不是為了換來一項商品；而是讓別人也有機會享用你所享用的。

咖啡發展能分為三波浪潮。第一波是強調方便、快捷的咖啡速食文化，以即溶咖啡（instant coffee）為主；第二波著重享受咖啡的經驗，以星巴克咖啡為首，特濃咖啡（espresso）和卡布奇諾（cappuccino）等咖啡紛紛涌現。以精品咖啡（specialty coffee）的名字登場，但過度的商業化，味道一式一樣；而第三波則著重生產的源頭，強調精品咖啡的獨特性、咖啡豆的原始味道，不同產地的咖啡豆因而備受注目。
