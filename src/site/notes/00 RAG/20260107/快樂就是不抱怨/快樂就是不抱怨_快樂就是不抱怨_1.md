# 快樂就是不抱怨


我們經常勸別人不要「執著」。執著是甚麼？  
「執著」就是抓住不放的意思。我們究竟抓住甚麼不放呢？  
抓住不放的可能是一個人、一段感情、一件物品，甚至是一件事情。  
但我們最抓住不放的，往往是自己的觀念和看法。
