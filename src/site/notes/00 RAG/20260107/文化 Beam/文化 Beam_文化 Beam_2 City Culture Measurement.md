# 建立高密度未來城市

## 2. 在港島建立三條文化帶，貫通中西區，灣仔及東區
- 綠色文化帶
- 歷史文化帶
- 海濱文化帶

## 3. 在十八區設立「社區文化館」
- 社區歷史館
- 社區藝術設施
- 圖書館 / 終身教育
- 創意市集
- 社區農場 / 農產品墟市
- 推動社區旅遊
- 推動多元文化

## 4. 活化工業區，建立「創意產業發展區」
- 教育及培訓機構
- 電影博物館和流行音樂博物館
- 創意產業孵化中心
- 與業界合作改裝工廠大廈作創意產業用途

## 5. 發展鄉郊文化
- 推廣行山文化
- 保育鄉村，推廣低碳生活文化
- 結合藝術，發展文化旅遊
