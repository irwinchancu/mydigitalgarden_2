# 克服衝突心境界
## 以靈性突破人性的衝突管理策略


問題在於，我們該如何做到寬恕，讓它具有能夠真正解決衝突的效果？在我看來，有五個步驟。

第一步：我應該允許自己感受到傷痛的存在。我不應該太快讓對方從對我造成的傷害當中脫罪或不理會它。我必須承認，對方對待我的方式讓我實在感到很心痛。

第二步：我需要憤怒，好讓自己在內心與對方拉開距離。如果我過度執著於自己內心悲傷的情緒，就無法去寬恕別人。如此一來，我就處於對方勢力範圍的控制之下。為了能夠寬恕別人，我需要內心的距離。我也需要憤怒，以便能夠接觸到自己內心的力量，不是用來報復的力量，而是讓自己有自信，不必依賴對方也可以獨立生活的力量。這力量也可以促使你，勇於讓對方意識到他所造成的傷害。受傷害的人通常覺得自己很無力。所以感受到憤怒是再度感受到自己和自己的力量的第一步。

第三步：我嘗試客觀地觀察發生過的事。這個衝突是怎麼形成的？是什麼言行傷害了我？那是誤會嗎？還是有一些與實質爭論無關的東西溜到對話裡了？對方之所以傷害我是因為他覺得自己處於劣勢嗎？或者他只是將自己童年時的挫折和傷害轉化、表現出來？我嘗試去理解到底發生了什麼事。當我能夠明白這個衝突時，我才能在衝突中堅守自己。而堅守自己是解決衝突的先決條件。

第四步：我寬恕對方，解開我與他之間的傷害連結，並將自己從這傷害所造成的負面能量中釋放出來。寬恕對我自己有好處。它清除掉對方在我身上所造成的負面力量。此外，寬恕的意思是，我把這個傷害行為的一切留在對方身上。如果我不寬恕，我就一直和他綁在一起，我就受他的勢力所影響。寬恕將我從對方的權力中釋放出來，解開我和他之間的束縛。於是我就能繼續自由地走自己的路。寬恕的意思不是我馬上跑過去擁抱對方。很有可能，我的心靈被傷得很重，雖然我原諒對方，但還沒有辦法忍受與對方接近。這時，我應該思考，在寬恕之後該如何經營我與他之間的關係。很可能我們之間的距離更大，但也有可能會形成更真誠、更開放的關係。寬恕的意思不一定是我忘記發生過的事，而是我不受它的影響，過去的事不再影響支配著我的行為和感覺，所以我能夠對它放手或留在對方身上。

第五步：將傷口變成珍珠。這意思是：從這個衝突也可以以產生一些好的結果。我們能夠找到新的解決方式，為我們之間的相處找到新的基礎。而且這個衝突使我接觸到自己的真實面，接觸到自己的能力。經由這個衝突，我為真正的自我以及心靈的新機會而敞開自己。這個衝突使我變得更謙卑。它讓我看到自己的黑暗面。我也學習接受對方和他敏感的一面。我不責怪他，也不責怪自己。我們愈來愈為對方開放自己。於是透過這個衝突，愛更加增長，而且這份愛將我和對方身上的一切都包容進去。

## 那知道自己在上帝面前擁有尊嚴的人，就不需要為自己的尊嚴擔憂。別人根本無法奪走我的尊嚴

「有人打你的右臉，連左臉也讓他打吧！」（馬太福音五章40節）攔臉的行為在猶太人的傳統中象徵的比較不是暴力，而是羞辱對方。他們不是以手掌，而是以手背打對方。這不是真正的打，而是以手背去碰觸對方的臉，這要表達的是：「你算什麼？我鄙視你。」那知道自己在上帝面前擁有尊嚴的人，就不需要為自己的尊嚴擔憂。別人根本無法奪走我的尊嚴，因為我內心的尊嚴是沒有人能夠奪走的。因此，我不必一直為自己的尊嚴搏鬥。我的本質就擁有尊嚴，沒有人能奪得走它。耶穌在這裡的要求，並不是示弱的表現，反而是強大的表現。那強大的人，內心平和，所以不害怕別人羞辱的言詞。而且他使那想羞辱他的人感到不安。也許他使對方看到自己的尊嚴。由於他不能奪走我的尊嚴，他會開始找尋並相信自己的尊嚴。

# 第三個例子：「有人拉你上法庭，要你的內衣，連外衣也給他吧！」（馬太福音五章40節）

在猶太人的生活習慣中，絕對不可以在法庭上讓對方爭得自己的外衣，因為在寒冷的夜間我需要拿外衣充當暖和的毯子。耶穌在這一點的意思是：那知道自己受上帝護祐的人，甚至連那夜間需要用來保暖的外衣都可以給出去。意識到自己受到保護，可以讓我在沒有保護的情況下參與對話。我不必躲在任何基本原則的防衛工事之後，我為對話中發生的一切開放自己。我不躲在厚厚的外衣之後，躲在我的角色、我的面具之後。我在人前展示出最真實的自己。只有這樣，對話才可以形成。只有這樣，才能有真誠的相遇。

# 第二種行為方式：我應該善待那些待我不好的人，即使他們不公平地對待我，我也要好好對待他們。藉此我為他們開啟一個能夠在其中思考自己的行為並最後改變它的空間。

在耶穌看來，「愛」實際上是一種充滿創意的解決方法，它引領人在衝突中找到方法脫離毫無出路的死角，我們有時候也會因為我們每個人都堅持自己的立場，而陷入這類衝突。耶穌要求我們愛敵人並不是一種苛求的做法，是一種以創意去解決衝突的方法。當我陷入一個使人激憤的衝突中時，如果能夠一再提醒自己：上帝也愛我的對手，這是很好的一件事。於是我會自己試著把他們當作耶穌基督的兄弟姐妹去愛他們。在這麼做的時候，耶穌在馬太福音中為門徒們所舉的例子很有幫助：「天父使太陽照好人，同樣也照壞人；降雨給行善的，也給作惡的」（馬太福音五章45節）

如果在一家公司裡解決了一個非常困難的衝突，可以掛起一幅圖畫來提醒大家這個共同的解決方案。在這種情況下，以幽默的方式畫出衝突各方以及他們達到協議狀況的一幅漫畫會是很合適的做法。或者可以擺出一個明顯易見的象徵物作為和好的記號：例如一棵和好樹，樹枝上面掛滿員工對於能夠達到這個解決方案所寫的感言，可以是表達感謝、許願，也可以是一些充滿希望和信心的話。這類的和好記號會將緩和、輕鬆和自在的氣氛帶入這個被解決的衝突裡。
