# 思考

很多華人信奉的唯物主義思想，乃來自百多年前，德國經濟學者馬克思（Karl Marx, 1818-1883），他把人類生存，訴諸物質的價值。把文明的建設，區分為經濟基礎（economic basis）和上層建築（superstructure）。前者指物質生活，後者才是精神。但是精神建基於物質，而精神的最上層乃是政治思想。

這即是說，人類歷世耕耘所得的一切人文建樹——文化、道德、倫理、公義、法律、文學、藝術等等——都由政治掛帥，要完全依循以經濟發展為綱的指導思想去操作。

這種唯物思想，與老子對道的關懷，風馬牛不能相及！

「唯物主義」將人的思想，拘禁於物質之「有」。老子卻是教人，要跨越物質之有，看到非物質之「無」，才能接近「道」。對道關懷，雖不等於對上帝關懷，但是老子既然接受，不可見的道存在之可能，即也尊重上帝存在之可能。

可是唯物主義所建於其上的「無神論」，從一開始便將上帝存在之可能性去除，也將道的超越性去掉。使這個宇宙，只剩下赤裸裸的物質操作和運算機率，道德和價值觀念只成為達成人類生存目的之手段！

不知道自己來源的人類，卻去否定創造主存在之可能。此即以局部知識去定義全部真理，以小知當作全知。此即以受造者身分，去扮演創造者角色，自己去掌控世界。

一、有一個「居善地」的基督：神子降世不嫌卑屈，生於貧寒之家，身軀骯髒馬槽。

二、有一個「心善淵」的基督：基督全無與世人爭競之心。

三、有一個「與善仁」的基督：基督對求助者從不拒避。他說：「你們白白的得來，也要白白的捨去。」

四、有一個「言善信」的基督：基督言談從來不出妄語。那些因為妒恨而殺害他的人，反而這樣形容他：「你是誠實的，什麼人你都不徇情面……乃是誠誠實實傳上帝的道！」他教訓門徒：「你們的話是就說是，不是就說不是。」

五、有一個「正善治」的基督：基督叫有權有位的人這樣做：「外邦人有君王為主治理他們，那掌管他們的稱為恩主。但你們不可這樣，你們裡頭為大的，倒要像年幼的；為首領（作領袖）的，倒要像服侍人的。」在離世前一晚，基督逐個、逐個為學生洗腳，並說：「我是你們的主（尊者）、你們的夫子（老師），尚且洗你們的腳，你們也當彼此洗腳。」

六、有一個「事善能」的基督：基督「不是要受人的服侍，乃是要服侍人。」

七、有一個「動善時」的基督：基督時時刻刻遵行上帝旨意，直至於死。門徒說他是「向本丟彼拉多，作過那美好見證的」。17本丟彼拉多是羅馬帝國，於26至36年間，派駐巴勒斯坦地的總督，管治猶太人。基督在他的治下被釘十字架。教會看基督的受害，是最重要的事情：「要捨命，作多人的贖價！」18

基督門徒，也在學習這種如水之善的生活。

第二個層面，無指精神上的「虛己」。

在物理的世界，需有空間，始能成物。同樣在精神的生活，要能納虛，始能有德。

如果填塞人心虛位的，只有各樣的物質，那麼這人的生活內容，很難在精神生活有所晉升。

要謙，要虛，必先能夠有所捨棄。要捨去的，是對於物質的持有、擁有、執有。要捨去的，是對權力的慾望。要捨去的，是自我傲慢之想求。

周代人從《易》學原理，得知自然世界有一個不變的定律：盈滿後必會虧缺、虧缺後又復盈滿。任何形式的有，都是短暫的經驗。

第三個層面，虛己的我，關懷嚮往宇宙之「道」。

在老子思想裡，若能回到無之境界，便可保持心靈對世物的獨立與自由，6便可接近道，便可得到生命的安頓。

這並不是說，無等同道。而是說，人若要接近道，便不能執著於物質之有——**姑無論是對世物之**

物品有了物質，才能成為可見的物品。譬如一個碗，我們所接觸到的，是那些用來製作碗的——陶泥，而不是那個構成碗狀的——空間。

可是如果沒有空間，碗又不是碗了！

「當其無」說出了，一個物品之所以能夠成為物品，全賴在製作過程中，有人運用了「非物質」的空間。

例如車輪、容器、房屋，除了具備物質的材料，還要包含空間。不過世人大多只關注到，物品的物質成分，沒關注到非物質的境界。

# 那麼掌權的人，要怎樣回復純樸心性？

乃是向道學習。

前面六句話，說出心性純樸者應有六種情操：

一、能抱一：「載營魄抱一，能無離乎？」指有健全身心。「載」為助語詞。「營魄」即魂魄。「抱一」即身心和諧。「無離」指精神體魄不相離。

二、如嬰兒：「專氣致柔，能如嬰兒乎？」指要思想純一。「氣」指生理。「專氣」即讓生理的本能聽任自然，如嬰兒般的純一自然。「柔」象徵不含私慾。⁵「嬰兒」正是反樸歸淳的象徵。

三、如玄覽：「滌除玄覽，能無疵乎？」指要內心光明如鏡。「覽」同「鑒」，即鏡子。「玄覽」指內心光明至極。「無疵」即沒有疵漏。

四、能無為：「愛民治國，能無為乎？」指無為而治。「無為」指不憑私心行事，不求自我作為。

五、能為雌：「天門開闔，能為雌乎？」指態度隨順，不持己見。「天門」指身體感官。「開闔」即開關，指感官正常活動。像女性（母性）一樣包容、傾聽、隨順自然規律。最厲害的領導者往往像水，看似柔弱，卻能穿透最堅硬的阻礙。

六、能無知：「明白四達，能無知乎？」指不墮機心。「知」指機心智巧。沒有機心，可大澈大悟，此即「明白四達」之意。

### 一、 融合的「重點」：以退為進的變革

變革領導需要強大的影響力，而老子的智慧正好能提供這種**「無聲的影響力」**：

- **「玄覽」與「願景」：** 變革領導者需要描繪未來。若內心不夠「玄覽」（光明無瑕），願景就會夾雜私心，變成割韭菜或虛榮的口號。**心淨，願景才真。**
    
- **「為雌」與「共創」：** 傳統領導是「我說你聽」；變革領導則是激發員工潛能。「為雌」的隨順態度，能創造一個安全的空間，讓部屬敢於嘗試、失敗與創新。
    
- **「無知」與「群體智慧」：** 當領導者放下一己之「機心（智巧）」，不再覺得自己最聰明，才能真正聽見基層的聲音，引發集體智慧的爆發。
    

### 二、 融合的「難點」：當「無為」遇到「KPI」

要把這套古老智慧放入現代企業的變革中，最難的莫過於以下三點：

1. **「抱一」vs. 「焦慮的分散」：** 現代變革講求快、狠、準，領導者每天被無數資訊轟炸。要維持身心和諧（抱一）而不被績效壓力（焦慮）帶走，這需要極強的定力。
    
2. **「無為」vs. 「主動干預」：** 變革往往需要推動力。如何做到「推動變革，卻不憑私心行事」（無為而治），這條線很難劃分。很多領導者一急，就會從「引導者」變成「獨裁者」。
    
3. **「如嬰兒」vs. 「社會化的武裝」：** 在爾虞我詐的商場或政壇，保持「嬰兒般的純一」會被視為軟弱。如何**「外圓內方」**——外在應對變革的複雜，內在保持嬰兒的純粹，是最大的修行。
