這些細節往往是老師在教員室抱怨「學生唔聽教」時，被忽略的底層邏輯：

| **範疇**   | **表面「大同」** | **隱藏「小異」的細節**                                                                         |
| -------- | ---------- | ------------------------------------------------------------------------------------- |
| **認知層面** | **能力薄弱**   | 其實是**「工作記憶（Working Memory）」不足*<br><br>*，或是對該學科的**「專用術語」**（如科學、經濟）產生認同障礙，<br>而非理解力差。   |
| **家庭層面** | **態度懶散**   | 可能是**「家庭空間匱乏」**（家裡沒桌子做功課）<br><br>或**「跨代壓力」**。<br><br>學生在校的懶散，往往是回家後過度承擔家務或照顧責任後的疲勞反應。 |
| **情感層面** | **動機不足**   | 存在**「習得性無助（Learned Helplessness）」<br><br><br>**。他們不是不想贏，而是怕輸得太難看，所以用「不在乎」來保護自尊。       |
| **學習習慣** | **不交功課**   | 背後是**「學習序列斷裂」**。<br><br><br>Band 2 學生常在某個年級（如中二升中三）斷了層，之後的課程對他而言如同天書，這不是懶，是看不懂。       |

## 二、 如何「打破」？（從行政到教學的介入）

要打破這些細節裡的魔鬼，不能單靠老師的熱誠，需要系統性的「機制」：

### 1. 數據與訪談結合：打破「認知盲區」

- **做法：** 建立「學生檔案 2.0」。除了成績，加入「學習障礙篩查」或「家庭環境標籤」。
    
- **例子：** 若發現某班 10 個學生沒交功課，5 個是因家庭環境，5 個是因基礎斷層。行政上應安排前者留在學校「課後自修室」，後者進入「補底班」，而非統一留堂處分。
    

### 2. 課程的「降維打擊」：打破「抽象屏障」

- **做法：** 推動 **Scaffolding（鷹架教學法）**。
    
- **具體操作：** 要求科組將抽象概念「圖像化」或「情境化」。例如，教理科概念時，先用生活例子（如遊戲規則）做類比，再進入公式。這能解決那些對抽象概念不感興趣的「小異」。
    

### 3. 微型成功感：打破「習得性無助」

- **做法：** 推行 **「小步子原則（Small Wins）」**。
    
- **具體操作：** 在評核中加入「努力分」或「進步獎」。打破「一次大考定生死」的傳統，改為頻繁、低難度的「微測驗」，讓能力薄弱的學生先嚐到「我能行」的滋味。
    

### 4. 師生關係的「非教學接觸」：打破「心理防線」

- **做法：** 副校長可帶頭推動「15 分鐘非學術對話」。
    
- **具體操作：** 讓班主任每週抽出時間，不談功課，只談興趣（如手遊、潮流）。當「小異」的細節（如學生的興趣、家裡的困難）被看見，學生對學校的歸屬感增加，動機自然會回升。

## 給副校長的建議語錄：

> 「老師們，我們不是要解決一個『懶散的群體』，而是要扶起一群『各有困頓的個體』。當我們把『唔想讀』拆解成『讀唔明』、『無位讀』或『驚讀錯』時，解決的方法自然呼之欲出。」

**這份細節清單是否符合您校內學生的實際情況？若您希望針對某個特定科目（如英文科或數學科）來深入探討，我也能為您提供更專門的對策。**




學習表現不理想，大多是因為動機不足、態度懶散、能力薄弱等幾項「大同」的因素；

但出現以上問題的個別學生可能只不過對抽象概念不感興趣、家庭環境惡劣、已有知識不足，這都可以是導致「大同」現象背後的「小異」問題。

假若不針對這些「小異」下工夫，解決方法往往只會事倍功半。「魔鬼在細節裏」就是這個道理。尤其涉及人的教育工作，課室內存在的差異絕非實驗室條件所能控制。
